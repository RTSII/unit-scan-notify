# MCP Server Integration

This folder is intended to contain or reference the magic-mcp server for local development and code intelligence.

## Usage

1. Clone or copy the magic-mcp server into this folder, or use it as a reference to your MCP server location.
2. To start the server, open a terminal in this folder and run:
   npm install
   npm start

For more details, see: https://github.com/21st-dev/magic-mcp#
